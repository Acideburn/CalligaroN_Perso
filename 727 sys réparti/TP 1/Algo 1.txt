Algo du calcul d'itération d'un mot dans un texte

Fct main (orchestre):
Docstring :
- Description détaillé

Entrée :
un fichier (type inconnu)
Sortie :
Affichage ou dictionnaire
Action :
Appeler la fonction _Ovrir_ sur le fichier
	recevoir une chaine
Appeler la fonction _Sub_Chaine_ sur la String
	recevoir une liste
Appeler la fonction _Build_dictionnaire_ sur la string
	Recevoir un dictionnaire 	

Fct ouvrir
Entrée :
un fichier (type a définir) prévoir des pilotes
Sortie :
Une chaine de caractère
Action :
Faire un pilote par type de fichier pour l'ouvrir et le lire.
prévoire une gestion d'erreur

Fct Sub_Chaine
Entrée :
Une chaine de caractère
Sortie :
Une liste
Action :
Subdiviser la chaine de caractère par différent caractère
-Retour chariot
-Espace
- Prévoir d'autre possibilité
Insérer chaque subdivision dans une liste.

Fct Build_Dictionnaire :
Entrée :
une liste
Stringortie :
un dictionnaire de type {var:int}
Action :
parcourir la liste par mot.
si une clé existe déjà avec le mot on incrémente la valeur
si aucune clé existe on la créé avec le mot en clé et la val=1
